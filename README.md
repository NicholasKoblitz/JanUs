# JanUs

## Overview

Welcome to the new way to create groups for online learning. With Janus a teacher is able to easily create a new course that their students are able to join and than assign those students to groups. This provides a faster and easier experience for both the teacher and the students.

The main features include:

- Separate student and teacher registeration.
- Individual course pages that display all students in the course along with the groups that student is assinged to.
- Teacher only group page to assign students to a perticular group
- Group specific chatroom for only the assigned students.


Come give JanUs a try to [janus.surge.sh](https://janus.surge.sh/)

---

## Technologies Uses

JanUs was built using Reactjs as the frontend framework with an Express.js backend the utilized a third party api from CometChat to design the chat group features of the site, and a custom api for the backend data.
